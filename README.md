# Spam⚠️ Detector for Email

## Usage:-

- Clone my repository.
- Open CMD in working directory.
- Run `App.py`.
- Go to the `http://127.0.0.1:5000` and test it. It is a local Flask App.
- Heroku App is available here:- [NLP App](https://spam-nlp-app.herokuapp.com/)
